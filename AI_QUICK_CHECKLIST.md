# AI Code Generation - Quick Checklist ✅

**USE THIS BEFORE CLAIMING ANY CODING TASK IS COMPLETE**

---

## Swift Array Generation

- [ ] Every array element has a comma (except last)
- [ ] Multi-line objects: comma AFTER closing `)`
- [ ] Pattern check: No `)` followed immediately by new element name
- [ ] Visual scan: Each element ends with `,` or is legitimately the last

---

## Syntax Verification

- [ ] Code follows language syntax rules (Swift/Python/etc)
- [ ] All brackets/parentheses properly closed
- [ ] All strings properly quoted
- [ ] All required semicolons/commas present

---

## Build Verification

- [ ] Code builds/compiles successfully
- [ ] No syntax errors reported
- [ ] No warnings about malformed code

---

## Self-Verification Statement

Before saying "Done":

> ✅ "I have verified all syntax including commas, built the code, and confirmed no errors."

If you can't honestly say this → **THE TASK IS NOT COMPLETE**

---

## Red Flags 🚩

STOP and re-check if you see:
- 🚩 "Expected ',' separator" error
- 🚩 "Consecutive statements" error  
- 🚩 Building fails after code generation
- 🚩 Syntax highlighter shows unexpected colors
- 🚩 You generated 50+ lines without testing

---

## Golden Rule

> **"Generate code → Verify syntax → Build test → THEN claim complete"**

Never skip verification. Never assume correctness. Always validate.
