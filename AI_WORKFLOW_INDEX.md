## 🚀 Quick Start Guide

### For Developers:

**Starting a new AI coding session:**
```bash
1. Open AI_SESSION_PROMPT.md
2. Copy the session template
3. Send to AI at session start
4. Keep AI_QUICK_CHECKLIST.md open
5. Reference SWIFT_ARRAY_TEMPLATES.md as needed
```

**When AI completes a task:**
```bash
1. Check AI_QUICK_CHECKLIST.md
2. Verify all checkboxes
3. Build the code
4. Confirm no errors
```

**After an incident:**
```bash
1. Document in AI_WORKFLOW_FEEDBACK.md
2. Update relevant templates
3. Add to checklist if needed
4. Review with team
```

## 🆘 Troubleshooting

→ Share with team
