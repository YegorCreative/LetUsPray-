# LetUsPray

LetUsPray is a SwiftUI iOS app designed to help people pray through Scripture one day at a time.

The app begins with a simple devotional experience called **30 Days in Proverbs**, where each day includes verse-by-verse prayers inspired by a chapter from the Book of Proverbs.

## Vision

LetUsPray helps users build a consistent prayer habit by turning Scripture into daily prayer, reflection, and spiritual growth.

## MVP Features

- Daily prayer screen
- 30 Days in Proverbs plan
- Verse-by-verse prayer format
- Save favorite prayers
- Mark days as prayed
- Simple, peaceful SwiftUI design
- Local data only for the first version

## Future Features

- Prayer journal
- Daily prayer reminders
- Prayer streaks
- More prayer plans
- Audio prayers
- Shareable prayer cards
- Church and group prayer plans

## Tech Stack

- Swift
- SwiftUI
- iOS
- Xcode
- Local static data
- UserDefaults / AppStorage for simple persistence

## Project Status

Early development.

## Goal

Create a calm, beautiful, Scripture-centered prayer app that helps people pray daily and grow closer to God.
