💡 KEY INSIGHT
═══════════════════════════════════════════════════════════════════════════════

  The app does NOT load JSON files at runtime.

  JSON → Script → Swift File → Compiled into App

  JSON files are for:
    • Reference
    • Version control
    • Making changes easy

  Swift file (PsalmsPrayerData.swift) is what the app uses.
